Bar Ilan Software Support - Chapter 1 Installing Responsa


    Before you can start using Bar Ilan Reponsa, you have to install it on your computer. This installation process, which takes a few minutes, is largely automatic; see the instructions below.

Installing Responsa
    To install Responsa:

1.    Turn on your computer.

2.    Insert the DVD into your DVD-ROM drive.

3.    Double-click My Computer in Windows.
 
5.    Double-click the icon of your DVD-ROM drive.

6.    Double-click the file SETUP.EXE

Bar Ilan Software Support - Zooming Options for Images

   

Some of the Bar Ilan Responsa databases include images – for example, the maps in the Talmudic Encyclopedia.

    You can change the size of these images, zoom in or out of them, etc., by clicking the appropriate icons or selecting the appropriate menu item under View, as described below.
 

          
 

Icon Function Using It

Page view icon
 		 Sets the magnification level of the image 1. Click the arrow next to the Page View box.

2. Click the magnification level that you want – more than 100% to increase magnification, and less than 100% to zoom out.
 

Percentage to zoom in or out

 Enables you to zoom in on or out of the image by a specified percentage 1. Click the arrow in the Bar IlanCD next to the zoom percentage box    .

2. Click the percentage by which you want to zoom in or out of the image, or enter it yourself.
3. Click one of the Zoom In icons       or one of the Zoom Out icons       (see below) to magnify or reduce the size of the image by the specified percentage.

 
Zoom in icon
Zoom out icon
 		 Zooms in on or out of the image by the percentage specified in the Zoom Percentage box. After setting the percentage by which to zoom in or out in the Zoom Percentage box, click the appropriate icon to zoom in or out.
  Zoom point in icon
  Zoom point out icon
 		 Zooms in or out of the image, positioning the specified part of the image at the center of screen. 1. Set the percentage by which to zoom in or out in the Zoom Percentage box
2. Click the appropriate icon, depending on whether you want to zoom in or out.
3. Point to the part of the image which you want to appear at the center of the screen, and click.
 
Zoom Rectangle icon Specifies a part of the image to enlarge and zooms in on that part of the image 1. Set the percentage by which to zoom in in the Zoom Percentage box
2. Click the Zoom Rectangle icon.
3. While holding down the left mouse button, click on a part of the image which you  want to enlarge.
4. Drag the mouse pointer to draw a rectangle around the part of the image you want to enlarge.
5. Release the mouse button when you are ready to display an enlarged view of this part of the image.
 
Select icon Enables you to select part of the image, which can then be copied, printed, or saved 1. Click the Select icon.
2. Drag the mouse pointer to draw a rectangle around the part of the image you want to select.
3. To print the selection, click File, then Print Selection. To copy the selection, in the Bar Ilan Responsa Project click Edit, then Copy Selection.
 
Pan icon Lets you drag the image up, down, right, or left 1. Click the Pan icon.

2. While holding down the left mouse button, click any part of the image.

3. Drag the mouse pointer to drag the image in the desired direction.

4. Release the mouse button when the image is positioned where you want it.
 
  Overview window Displays a miniature overview of the entire image which appears in the image window. The part of the image currently displayed in the image window is surrounded by a dotted rectangle in the Overview window. Click the Overview window icon. An overview of the image will be displayed in the upper-right corner of the screen.
Rotate icon Rotates the image 90 degrees clockwise. Clicking the icon repeatedly causes the image to rotate another 90 degrees each time the icon is clicked Click the Rotate icon
Flip Horizontal icon Displays a mirror image of the image on screen Click the Flip Horizontal icon
  Flip Vertical icon Displays a mirror image of the image on screen, rotated 180 degrees Clip the Flip Vertical icon
Copy icon Copies the selected part of the image as selected above Click the Copy icon.
Print icon Prints the image as displayed on screen Click the Print icon
Help icon Displays help on all image-related functions Click the Help icon.
